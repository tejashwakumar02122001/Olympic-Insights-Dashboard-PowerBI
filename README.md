# 🚀 Features

- 📈 Olympic participation trends (1896–2016)
- 🌍 Country-wise medal analysis
- 🥇 Gold, Silver & Bronze medal insights
- 👤 Top athletes analysis
- ⚽ Sport-wise performance
- 🚻 Gender participation trends
- 🏠 Host nation analysis
- 🎛️ Interactive slicers & filters
