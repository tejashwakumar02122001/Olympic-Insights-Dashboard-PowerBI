## 🚀 Features

- 📈 Olympic trends from **1896–2016**
- 🌍 Country-wise medal analysis
- 🥇 Gold, Silver & Bronze medal breakdown
- 👨‍🎓 Athlete performance analysis
- ⚽ Sport-wise insights
- 🚻 Gender participation trends
- 🏠 Host nation performance
- 🎛️ Interactive filters & slicers

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- CSV Dataset
